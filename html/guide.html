GUIDE

* 기본 규칙

  • class 이름의 단어구분은 "-" 으로 한다. 예시 ) tab-wrap
  • scss 이름은 html 파일명과 동일
  • 지금 이 규칙 리스트도 활용하세요. 리스트 양식입니다.
  • 기본 글씨는 크기 14px, 굵기 400, line-height 1입니다. line-height는 px 이 아닌 배수로 쓰시는 게 반응형에서 더 편리합니다.
  • 가이드로 정해져 있는 부분이라도 더 좋은 아이디어가 있으시다면 언제라도 변경하셔도 좋습니다~!!
  • JS는 제이쿼리와 기본 바닐라 모두 허용합니다.
